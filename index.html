Your data will be saved in the cloud — survives clearing history, switching phones, everything. One-time setup, takes about 3 minutes.

1
Go to supabase.com → Sign up free → click New project. Pick any name, set a password, choose a region close to you (e.g. EU West).
2
Once the project loads, open the SQL Editor (left sidebar) and paste + run this:create table habits ( id bigint primary key, name text, emoji text, type text, active boolean, sort_order int default 0 ); create table logs ( date text primary key, data jsonb );
3
Go to Project Settings → API (bottom of left sidebar). Copy your Project URL and anon / public key and paste them below.
